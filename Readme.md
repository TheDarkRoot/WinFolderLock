# Installation

> ***Follow these steps.***
```
$ Download software.
$ Right click: Run as an administrator.
$ Press a key to continue.

- Default Folder: WinFolder
- Default password: Psswrd

# Change password:
$ WinFolderLock
$ Right Click
$ Edit
$ Change "Psswrd" in line 54
$ Save
$ Exit
```
